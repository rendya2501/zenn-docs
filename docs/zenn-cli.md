# zenn-cli

## コマンド

| 操作 | コマンド |
| :--- | :------- |
| 記事を作成 | `npx zenn new:article` |
| プレビュー | `npx zenn preview` |

> **注意:** コマンドはすべて `zenn-docsリポジトリ` のルートで実行すること
